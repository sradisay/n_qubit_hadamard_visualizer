# n-Qubit Hadamard Visualizer

A lightweight, vanilla JavaScript tool to visualize the phase patterns generated by applying an $n$-qubit Hadamard transform to a specific basis state.
Visualizes the resulting waveform from integer parameters n and x.

## Demo

1.  Try it out at <a href="https://quantum.sradisay.dev" target="_blank" rel="noopener noreferrer">https://quantum.sradisay.dev</a>

2.  Enter the number of qubits ($n$) and the integer state ($x$).
3.  View the generated square wave graph representing the phase distribution.
   
##  How it Works

The visualizer implements the definition of the Hadamard transformation acting on a basis state $|x\rangle$:

$$H^{\otimes n} |x\rangle = \frac{1}{\sqrt{2^n}} \sum_{y=0}^{2^n - 1} (-1)^{x \cdot y} |y\rangle$$

The script calculates the **bitwise inner product** ($x \cdot y$) for every possible output state $y$.

  * If the parity is **even** ($0$), the graph draws a high/positive signal.
  * If the parity is **odd** ($1$), the graph draws a low/negative signal.

##  code snippet

The core logic relies on finding the modulo 2 of the set bits in the conjunction of $x$ and $y$:

```javascript
function bitwise_inner_product(x, y) {
    let n = x & y;
    let count = 0;
    while (n !== 0) {
        n &= (n - 1);
        count++;
    }
    return count % 2;
}
```
